pydpeet.SequenceOverviewConfig#

class pydpeet.SequenceOverviewConfig[source]#

Bases: object

Container class providing predefined sequence overview configurations.

This class holds static factory calls for common sequence overview configuration types. Access predefined configs via class attributes, or create custom configs using the sequence_overview_config_wrapper function.
